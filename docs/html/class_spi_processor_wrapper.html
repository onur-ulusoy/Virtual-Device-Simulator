std::string SpiProcessorWrapper::requestReadLine ( const std::string &  writeLine)

Requests a read_line from the spi processor.

This method takes a string representing the write_line to be sent to spi processor, writes the command to the "in" file, and waits for a response in the "out" file. If the response is a valid spi_read line or the string "TERMINAT", the method returns the response.

Parameters
writeLineThe write command to be sent to spi processor.
Returns
A string representing the read response from spi processor.
