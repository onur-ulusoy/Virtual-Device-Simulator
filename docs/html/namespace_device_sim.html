◆ receive_command()

int DeviceSim::receive_command ( enum command_request  req,
string  devType,
fstream &  receiver,
string &  _command,
ofstream &  _com 
)

Receives commands that are sent by tester with the specified request and performs the necessary operations.

Parameters
reqThe type of command request. Can be either ONESHOT or RECURSIVE. If commands received from text file, RECURSIVE mode runs.
devTypeThe type of device being processed. Can be one of "gpio", "spi", "i2c", "ethernet", "usart", "uart", "can".
receiverStream object for reading the command from command text.
_commandString to store the received command.
_comStream object for writing responses to communication register.
Returns
-1 if the command is to disconnect the device, 0 if the command is to process the device or to execute nothing.
