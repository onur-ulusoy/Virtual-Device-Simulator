Detailed Description

Runs the compiling and running processes of the system. 

The script implements a program for compiling and running source files for a driver and a tester concurrently by running different processes.
It also allows the user to choose which device to simulate among several options.

◆ main()

def start.main ( )

Main function for the program.

This function provides three options to the user: compile, run, and compile_run. Depending on the user's choice, the appropriate action is performed. The function uses threads to perform actions in parallel and communicates with other threads using a file. It also creates proper directories for virtual devices that will be simulated.
