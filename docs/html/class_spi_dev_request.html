Detailed Description

A class to process SPI device requests from raw data.

This class is responsible for processing raw SPI device request data, converting it to JSON format, parsing and processing the JSON data, and saving the processed data as a new JSON file. It also provides methods to access the SpiDevEntry objects created from the parsed and processed JSON data.

◆ rawToJson()

void SpiDevRequest::rawToJson ( )

Convert the raw SPI device request data to JSON format and save it as a new file.

This method reads the input file (as in this context such as SPI_A, consists of spi_write lines) line by line, extracts relevant information, and constructs a JSON object representing the SPI device request. The JSON object is then saved to a new file with the same name as the input file but with a ".json" extension.
