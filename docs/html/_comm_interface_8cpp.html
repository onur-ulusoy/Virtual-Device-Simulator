Detailed Description

Contains the Publisher and Subscriber classes for communication between processes using ZeroMQ.

Provides a C++ implementation of the ZeroMQ publisher-subscriber communication pattern. The Publisher class sends messages to one or more subscribers using a publisher socket, and the Subscriber class receives messages from a publisher using a subscriber socket.
