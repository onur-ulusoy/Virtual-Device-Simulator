std::string Subscriber::receive ( int  timeout = -1)

Receives a message using the Subscriber object's socket and returns the received message. The received message is deserialized using the msgpack library and printed to the console. If no message is received or a timeout occurs, an empty string is returned.

Parameters
timeoutThe maximum time to wait for a message, in milliseconds. -1 (default) means wait indefinitely.
Returns
The received message as a std::string or an empty string if no message is received or a timeout occurs.
