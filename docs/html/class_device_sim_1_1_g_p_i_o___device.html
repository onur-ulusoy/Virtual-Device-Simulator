◆ device_close()

void DeviceSim::GPIO_Device::device_close ( )

Closes the file stream 'fd' associated with the device.

If the file stream 'fd' is open, it closes the stream and outputs a message indicating that the file stream is closed successfully. If the file stream 'fd' is not open, it outputs a message indicating that the file stream is already not open.
